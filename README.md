# Portfólio — Atividade Prática (Fundamentos da Programação Web)

Este projeto é um portfólio pessoal desenvolvido conforme as instruções da atividade prática.

## Objetivo
Construir um portfólio online usando **HTML5**, **CSS3** e **JavaScript** (sem frameworks), contendo 4 páginas com navegação fixa entre elas e um formulário de contato validado por JavaScript.

## Estrutura do site
O site foi implementado no modelo de **páginas separadas**, conforme permitido na atividade. Todas as páginas possuem o mesmo menu fixo no topo.

### Páginas
- **Sobre mim** — `index.html`
- **Formação** — `formacao.html`
- **Portfólio** — `portfolio.html`
- **Contato** — `contato.html`

## Requisitos atendidos
- **4 páginas separadas**, interligadas por menu visível em todas elas.
- **HTML5 + CSS3 + JavaScript** sem uso de frameworks.
- **Formulário de contato** com campos nome, e‑mail e mensagem.
- **Validação obrigatória em JavaScript** (campos obrigatórios e e‑mail válido).
- **Simulação de envio** com limpeza do formulário e mensagem de confirmação.
- **Interações simples**: menu responsivo e alternância de tema (claro/escuro).

## Funcionalidades em JavaScript
- **Menu responsivo** para telas menores.
- **Tema claro/escuro** com alternância por botão.
- **Validação do formulário** com mensagens de erro e sucesso.

## Arquivos principais
- `index.html` — Sobre mim
- `formacao.html` — Formação
- `portfolio.html` — Portfólio
- `contato.html` — Contato
- `styles.css` — Estilos do site
- `script.js` — Funcionalidades e validações

## Observações
- O envio do formulário é **simulado**, conforme solicitado no enunciado.
- Todo o conteúdo e layout foram feitos manualmente, sem bibliotecas externas.
